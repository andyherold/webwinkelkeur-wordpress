Installatieinstructies
----------------------

    1. Plaats de map "webwinkelkeur" in de map "wp-content/plugins" op de
       webserver.

    2. De plugin kan nu worden geactiveerd in de administratieinterface. Ga
       daarvoor naar Plugins en klik op Activeren bij de Webwinkelkeur plugin.

    3. Klik op Instellingen om de plugin te configureren.

    4. Controleer of de sidebar op uw site wordt getoond.
